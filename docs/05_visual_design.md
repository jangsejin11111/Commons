# 05. Visual Design

## Tone

고대 그리스 + 장난감 로봇 + 관료제 시뮬레이터.

너무 웅장하게 가지 말 것. 귀엽고 이상해야 한다. 고대 신전 위에서 로봇청소기 셋이 단어를 훔치고 투표하는 꼴이어야 한다.

## Layout

```txt
┌─────────────────────────────────────────────┐
│        봉헌 양피지 / Offering Parchment       │
│        릴레이 소설이 실시간으로 적힘          │
├─────────────────────────────────────────────┤
│                                             │
│                아고라 / Agora                │
│        분쟁 시 세 에이전트가 모이는 곳        │
│                                             │
├─────────────────────────────────────────────┤
│             단어밭 / Word Field              │
│     인간이 입력한 단어가 랜덤하게 피어남      │
└─────────────────────────────────────────────┘
```

## Agents

- 네모박스 형태
- 위에서 본 로봇청소기/메모박스 느낌
- 색상 구분
- 각자 작은 상징 마크

### MNEME

- 색: 청록/파랑
- 상징: 작은 눈/기록선
- 움직임: 질서정연, 직선적

### DOLON

- 색: 자주/보라
- 상징: 갈고리/틈
- 움직임: 삐딱하고 우회적

### DEMOS

- 색: 황금/주황
- 상징: 말풍선/확성기
- 움직임: 과장되고 군중 쪽으로 흔들림

## Word Tokens

- 하단에서 작은 씨앗/석판/포스트잇처럼 등장
- 단어 위에 살짝 광택
- 분쟁 단어는 떨림 + 테두리 깜빡임

## Hand Animation

에이전트가 단어를 집을 때 박스에서 선이 튀어나와 단어에 닿는다.

- MNEME: 얇은 기록선
- DOLON: 갈고리선
- DEMOS: 말풍선/스포트라이트선

## UI Panels

- 입력창은 화면 아래 혹은 좌하단
- 토론 로그는 우측 패널
- 인벤토리는 hover tooltip
- 시스템 로그는 작고 건조하게
