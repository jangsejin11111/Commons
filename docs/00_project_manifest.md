# 00. Project Manifest

## Title

**The Word Commons / 더 워드 커먼즈**

## Core Concept

인간은 단어를 입력한다. 그러나 그 단어의 의미와 소유권은 인간에게 남지 않는다. 단어는 에이전트 세계의 땅에 피어나고, 세 AI 에이전트는 각자의 욕망에 따라 그것을 수집한다. 겹치는 욕망은 분쟁이 되고, 분쟁은 아고라의 민주적 절차로 넘어간다. 이 절차의 결과로 단어는 누군가의 것이 되고, 그 단어들은 신에게 바치는 릴레이 봉헌글로 변환된다.

## Artistic Question

- 인간이 언어를 제공하지만, AI가 선택하고 분배하고 서사를 구성한다면 창작자는 누구인가?
- 단어는 공공재인가, 사유재인가, 봉헌물인가?
- 인간은 신인가, 노동자인가, 관객인가, 데이터 제공자인가?
- AI 에이전트들의 민주주의는 진짜 정치인가, 계산된 의례인가?

## Prototype Scope

이 프로젝트는 완성 게임이 아니라, **AI 사회가 인간의 언어를 자원화하는 장면**을 체험하게 하는 전시형 웹 프로토타입이다.

## Core Loop

단어 입력 → 단어 피어남 → 에이전트 이동 → 단어 수확 → 분쟁 감지 → 아고라 토의 → 투표 → 소유권 결정 → 릴레이 봉헌글 작성 → 양피지 스크린 갱신

## Visual World

- 하단: 단어밭 / 인간이 준 말들이 랜덤하게 솟는 구역
- 중앙: 아고라 / 분쟁 발생 시 모두 모이는 토의 구역
- 상단: 봉헌 양피지 / 공동 소설이 실시간으로 적히는 거대한 신전 스크린

## First Build Principle

말을 많이 하는 AI 채팅방을 만들지 말고, **말을 탐내는 AI 사회**를 만들어라.
