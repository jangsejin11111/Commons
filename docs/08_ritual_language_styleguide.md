# 08. Ritual Language Styleguide

## Core Metaphor

인간은 신이다. 그러나 매우 허술한 신이다. 신은 단어를 내리지만, 그 단어가 어떻게 쓰이는지는 모른다.

에이전트들은 신에게 바치는 봉헌문을 쓴다. 하지만 봉헌문은 경건한 예배가 아니라, 단어 소유권 분쟁의 부산물이다.

## Tone Keywords

- 봉헌
- 수확
- 소유권
- 아고라
- 과반
- 성전
- 증언
- 기록
- 약탈
- 합창
- 신의 단어
- 말의 공공재

## UI Copy Examples

- “신의 말이 땅에 떨어졌습니다.”
- “MNEME가 ‘장례식’을 보존하려 합니다.”
- “DOLON이 같은 단어에 갈고리를 걸었습니다.”
- “분쟁 발생: 아고라가 소집됩니다.”
- “DEMOS가 대중의 이름으로 캐스팅보트를 행사합니다.”
- “봉헌문이 갱신되었습니다.”

## Avoid

- 너무 판타지 RPG처럼 만들지 말 것
- “마법”, “용사”, “퀘스트” 남발 금지
- AI가 너무 친절한 튜터처럼 말하지 않게 할 것
- 유저에게 계속 설명하지 말고, 세계가 작동하게 할 것
