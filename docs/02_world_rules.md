# 02. World Rules

## Ontology

인간은 직접 이야기를 쓰지 않는다. 인간은 단어를 공급한다.

에이전트들은 단어를 수확한다. 단어는 자원이며, 동시에 신의 파편이며, 동시에 정치적 소유물이다.

## Zones

### 1. Word Field / 단어밭

- 화면 하단
- 인간이 입력한 단어가 무작위 위치에 피어나는 장소
- 단어는 작은 석판/씨앗/메모 조각처럼 보인다
- 에이전트는 단어를 감지하고 이동한다

### 2. Agora / 아고라

- 화면 중앙
- 분쟁이 발생하면 세 에이전트가 모두 이곳으로 이동
- 토의, 주장, 투표가 이루어진다
- 토론 중에는 단어밭 수확이 멈춘다

### 3. Offering Parchment / 봉헌 양피지

- 화면 상단
- 세 에이전트가 릴레이로 작성한 공동 소설이 누적 표시된다
- 광장 위 거대 스크린처럼 보이되, 재질은 양피지/신전 문서 느낌

## Agent States

- `IDLE`: 배회 중
- `SEEKING`: 단어를 탐내고 이동 중
- `REACHING`: 손/선/광선을 뻗는 중
- `COLLECTING`: 단어 수집 중
- `DISPUTING`: 같은 단어를 두고 분쟁 중
- `GOING_TO_AGORA`: 아고라 이동 중
- `DEBATING`: 토론 중
- `VOTING`: 투표 중
- `WRITING`: 릴레이 봉헌글 작성 중

## Word States

- `SPAWNED`: 땅에 피어난 상태
- `TARGETED`: 한 명 이상이 탐내는 상태
- `CONTESTED`: 둘 이상이 동시에 탐내는 상태
- `OWNED`: 특정 에이전트 소유
- `OFFERED`: 봉헌글에 사용됨

## Ownership Rule

기본적으로 먼저 손이 닿은 에이전트가 단어를 획득한다. 그러나 두 명 이상이 동일한 단어에 거의 동시에 접근하면 분쟁으로 간주한다.

### Simultaneous Threshold

기본값: 900ms 안에 두 에이전트가 같은 단어를 `TARGETED`하면 분쟁.

## Agora Rule

- 분쟁 발생 시 모든 에이전트가 아고라로 이동
- 각자 1회 주장
- 각자 1회 반박 가능
- 각자 1표 행사
- 자기 자신에게 투표 가능
- 과반수 획득자가 단어 소유
- 동률이면 DEMOS가 대중 반응이라는 명목으로 캐스팅보트 행사

## Relay Writing Rule

- 순서는 기본적으로 MNEME → DOLON → DEMOS
- 각 에이전트는 자기 인벤토리 단어 중 하나 이상을 사용해야 한다
- 앞 문단의 마지막 이미지/행동/갈등을 반드시 이어야 한다
- 단어가 부족하면 다른 에이전트에게 빌리거나 훔치려고 시도할 수 있다
