# Claude Code에 처음 붙여넣을 메시지

아래 프로젝트는 `<The Word Commons / 더 워드 커먼즈>`라는 2D 멀티에이전트 웹 프로토타입입니다. 먼저 `CLAUDE.md`를 읽고, 그 다음 `docs/`와 `content/agents/`의 문서를 모두 확인한 뒤 구현을 시작해주세요.

우선순위는 다음입니다.

1. `npm install` / `npm run dev`가 가능한 Vite + React + TypeScript 프로젝트로 정리
2. Phaser 2D 월드 구현: 상단 봉헌 양피지, 중앙 아고라, 하단 단어밭
3. 세 에이전트를 색깔 있는 네모박스로 표시하고 배회/이동 구현
4. 인간이 입력한 단어가 하단 단어밭에 랜덤하게 피어나게 구현
5. 에이전트가 단어 취향에 따라 목표 단어를 선택하고, 손/선 애니메이션으로 수확하게 구현
6. 수확된 단어는 사라지고, 에이전트 hover 시 인벤토리 표시
7. 두 에이전트 이상이 같은 단어를 동시에 탐내면 아고라 분쟁 상태로 전환
8. mock LLM으로 토론/투표/봉헌글 작성 루프 구현
9. 실제 LLM Provider를 나중에 붙일 수 있도록 `src/llm/providers` 구조 유지

처음부터 실제 API를 강제하지 말고 mockProvider로 안정적인 발표용 MVP를 먼저 완성해주세요. 단어 수집 → 분쟁 → 아고라 투표 → 릴레이 봉헌글 작성이 한 번에 시연되면 성공입니다.
